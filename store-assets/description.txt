简短描述：
QR Code 助手 - 快速生成和识别二维码的 Chrome 扩展

详细描述：
QR Code 助手是一个功能强大的 Chrome 扩展，帮助您快速生成和识别二维码。

主要功能：
✨ 生成二维码
- 支持自定义二维码内容
- 可自定义二维码大小、边距
- 支持添加自定义标题
- 一键获取当前标签页 URL 并生成二维码
- 支持下载生成的二维码图片

📷 识别二维码
- 支持从图片文件中识别二维码
- 显示识别历史记录
- 支持清空历史记录

🎨 个性化设置
- 可自定义标题文字大小和颜色
- 简洁美观的界面设计
- 支持深色/浅色主题

使用场景：
- 快速分享网页链接
- 移动设备和桌面设备之间传递信息
- 识别图片中的二维码内容
- 生成自定义内容的二维码

隐私说明：
- 所有操作均在本地完成，不会上传任何数据
- 仅需要最基本的权限
- 完全开源，代码透明
